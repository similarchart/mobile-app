### 주요기능

- 8, 16, 32, 64, 128일 치의 다양한 비교 결과 제공
- 한국주식, 미국주식지원
- 한국어, 영어 지원
- 각 종목별 차트평가 제공
- 랭킹페이지, 검증페이지 제공
- 네이버증권, 야후증권 연동
- 사용자가 직접 차트를 그릴 수 있는 '드로잉검색', '패턴검색' 지원
